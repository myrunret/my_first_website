Discover how this site was built

A small Flask app with a big aim.

Purpose

This project combines multi-page scrolling interface, a little data visualisation, and accessible summarization of scholar research, so visitors can take it from here and bring new knowledge to their everyday life.

Data & privacy

Production numbers are served from a local SQLite database checked into the repo for reproducibility. The site also uses anonymous engagement tracking for coursework analysis: page views, section views, pledges, and contact submissions are stored with a random session ID.

Analytics events do not store IP addresses, browser user-agent strings, names, emails, or message text. Contact and pledge forms store the information visitors intentionally submit, separate from the anonymous event log.

Technology stack

Python Flask
A lightweight web framework for building the server-side of the website, handling routing, and serving HTML templates.
Front-end
Semantic HTML, one stylesheet (styles.css), and vanilla JavaScript (script.js) for scroll progress, intersection observers, canvas charts, and the pledge UI.
Fonts and Icons
Playfair Display, DM Sans, and DM Mono loaded from Google Fonts.
Icons from Font Awesome Free.
Tooling & AI
AI agents assisted with creation of the website, including generation of visuals. All AI-written code passes human review and the author remains responsible for accuracy of data and citations (see Data & sources).
